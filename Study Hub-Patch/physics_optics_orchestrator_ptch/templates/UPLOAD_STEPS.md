# Upload Steps

## Recommended upload path
1. Unzip the final merged pack.
2. Copy `Physics/optics/` into your Study Hub content folder.
3. Start backend / topic discovery.
4. Verify topic appears in parent and student views.
5. Open each page and confirm:
   - blocks render
   - helper notes render
   - related pages work
   - question bank is attached

## Optional path
If your branch supports content-pack import cleanly:
- import the generated `content-pack.json`
- verify assets and page refs are preserved

## Post-upload checks
- Topic visible in Physics
- 8–12 core pages visible
- Revision/remediation pages visible
- Question library available
- No 404 on page load
